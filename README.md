# Simon-Says-Game
This is a memory game in which the player has to remember a sequence of colors which increases in each level. 
Responsive frontend using HTML, CSS and JavaScript. Hosted the project using Netlify.
Implemented event-driven DOM logic to manage game state, scoring system and color animations.
Game score and highest scores are tracked and displayed.
Flashes of color tiles are achieved by classList property.
Boolean flags help to start/stop the game on specific conditions.
Made the game interface mobile-friendly using media-queries.
